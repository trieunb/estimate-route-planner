The instruction assumes that you already install Wordpress site.

0. Requirements check:
  - PHP >= 5.4
  - MySQL >= 5.4
  - Wordpress >= 4.2
  - The apache user(default is www-data or nobody) is able to read/write to the plugin directories,
    such as: tmp, log, uploads

1. Install PHP OAuth
  - Ubuntu/Debian: sudo apt-get install php5-oauth
  - Other: contact me if needed

2. Restart Apache and go to wordpress admin

3. Download and unzip plugin source to wp-content/plugins
    Expected result: <ROOT>/wp-content/plugins/estimate-route-planner

4. Active the plugin

5. Click to plugin menu (ER Planner Pro), it  will ask for connect with Quickbooks.

6. If connect successfully, click to "Settings" and then click "Sync Now".

7. Active cron job(every 5 minutes) for synchronize data, run this command:
  (crontab -l ; echo "*/5 * * * * /usr/bin/php /<path-to-root>/wp-content/plugins/estimate-route-planner/schedule/sync.php") | crontab -
  Verify cron job added: crontab -l
